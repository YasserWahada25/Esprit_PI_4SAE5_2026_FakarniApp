# Install Blue Ocean Plugin for Pipeline Visualization

## Steps:

1. Go to Jenkins: http://localhost:8090

2. Navigate to: Manage Jenkins → Plugins → Available plugins

3. Search for: "Blue Ocean"

4. Select these plugins:
   - Blue Ocean
   - Blue Ocean Pipeline Editor
   
5. Click: Install (without restart if possible, or restart if needed)

6. After installation, you'll see a "Open Blue Ocean" link in the sidebar

7. Click "Open Blue Ocean" to see your pipelines with beautiful visualization

## Alternative: Install via Jenkins CLI or Docker

If you're running Jenkins in Docker, you can add this to your Jenkins setup:

```bash
docker exec jenkins-container jenkins-plugin-cli --plugins blueocean:latest
```

## Access Blue Ocean:

After installation:
- URL: http://localhost:8090/blue
- Or click "Open Blue Ocean" in Jenkins sidebar

✅ Visual pipeline flow
✅ Real-time stage progress
