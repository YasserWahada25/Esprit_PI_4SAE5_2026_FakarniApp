┌────────────────────────────────────────────────────────────────────────────┐
│                                                                            │
│  Problème : Le frontend ne démarre pas                                    │
│  Solution :                                                                │
│    1. docker compose logs -f frontend                                     │
│    2. docker compose build --no-cache frontend                            │
│    3. docker compose up -d frontend                                       │
│                                                                            │
│  Problème : Les services ne se connectent pas                             │
│  Solution :                                                                │
│    1. Vérifier Eureka : http://localhost:8762                             │
│    2. Attendre 2-3 minutes pour l'enregistrement                          │
│    3. docker compose restart                                              │
│                                                                            │
│  Problème : Base de données non accessible                                │
│  Solution :                                                                │
│    1. docker compose ps (vérifier "healthy")                              │
│    2. docker compose restart db-tracking                                  │
│                                                                            │
│  Problème : Tout est cassé                                                │
│  Solution :                                                                │
│    1. clean-all.bat                                                        │
│    2. fix-all-issues.bat                                                   │
