# Create Docker Hub Credentials in Jenkins

## Steps:

1. Open Jenkins: http://localhost:8085

2. Navigate to: Manage Jenkins → Credentials

3. Click on: System → Global credentials (unrestricted)

4. Click: + Add Credentials

5. Fill in the form:
   - Kind: Username with password
   - Scope: Global (Jenkins, nodes, items, all child items, etc)
   - Username: didou2505
   - Password: [Your Docker Hub password]
   - ID: dockerhub-credentials
   - Description: Docker Hub credentials for didou2505

6. Click: Create

## Important:
- The ID MUST be exactly: dockerhub-credentials
- This is what the Jenkinsfiles reference
- Username MUST be: didou2505

## Verify:
After creation, you should see:
- ID: dockerhub-credentials
- Description: Docker Hub credentials for didou2505
- Kind: Username with password
