# How to View Pipeline Code in Jenkins UI

## Method 1: Pipeline Syntax / Replay (View Code)

1. Go to Jenkins: http://localhost:8085

2. Open your job: user-service-CI

3. Click on a build number (e.g., #6)

4. In the left sidebar, click: "Replay"
   - This shows the FULL Jenkinsfile code
   - You can edit and test changes here
   - Click "Run" to test modified code

## Method 2: Pipeline Steps View

1. Go to: http://localhost:8085/job/user-service-CI/

2. Click on a build number

3. Click: "Pipeline Steps"
   - Shows each step executed
   - Shows the code for each step

## Method 3: View in Job Configuration

1. Go to: http://localhost:8085/job/user-service-CI/configure

2. Scroll down to: Pipeline section

3. You'll see:
   - Definition: Pipeline script from SCM
   - SCM: Git
   - Repository URL: Your GitHub repo
   - Script Path: backend/User-Service/Jenkinsfile

4. Note: The actual code is in GitHub, but you can see the path

## Method 4: Blue Ocean Pipeline Editor (Best for Visualization)

1. Install Blue Ocean plugin (if not installed)

2. Go to: http://localhost:8085/blue

3. Click on: user-service-CI

4. Click: "Configure" or the pencil icon

5. You'll see a visual pipeline editor showing:
   - All stages
   - All steps within each stage
   - Can switch to "Code" view to see Jenkinsfile

## Method 5: Direct Code View (After Build)

1. Go to build: http://localhost:8085/job/user-service-CI/6/

2. Click: "Pipeline Overview" or "Pipeline Steps"

3. Each stage shows the code that was executed

## Best Option for Your Need:

Use "Replay" feature:
- Shows complete Jenkinsfile code
- Allows editing for testing
- Shows exactly what Jenkins executed

Steps:
1. http://localhost:8085/job/user-service-CI/
2. Click latest build (#6)
3. Click "Replay" in left sidebar
4. See full pipeline code!
